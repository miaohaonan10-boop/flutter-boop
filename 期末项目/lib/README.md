# 济宁家乡旅行推荐 / 지닝 고향 여행 추천

## 项目简介

本项目是基于 Python Tkinter 开发的双语（中文/韩文）旅游推荐应用，以山东省济宁市为主题，向用户介绍济宁的历史文化、著名景点、特色美食以及推荐旅游路线。

通过简洁直观的图形界面，帮助国内外游客了解济宁这座拥有深厚儒家文化底蕴的历史文化名城。

---

## 功能介绍

### 首页（Home）
- 济宁市基本信息介绍
- 城市特色展示
- 中韩双语城市简介

### 景点推荐（Attractions）
包含济宁代表性旅游景点：

- 曲阜三孔（孔庙、孔府、孔林）
- 邹城孟庙孟府
- 微山湖旅游区
- 水泊梁山风景区
- 太白楼

每个景点均提供：
- 中文介绍
- 韩文介绍
- 景区标签
- 必看景点推荐

### 美食推荐（Foods）

展示济宁特色传统美食：

- 甏肉干饭
- 孔府菜
- 济宁夹饼
- 糁汤
- 玉堂酱菜

包含中韩双语介绍及特色说明。

### 旅游路线推荐（Routes）

提供不同天数的旅行方案：

#### 一日文化精华游
- 曲阜三孔
- 孔府菜
- 孟庙孟府
- 太白楼

#### 两日深度体验游
- 三孔
- 尼山圣境
- 微山湖
- 甏肉干饭

#### 三日全景游
- 三孔
- 尼山圣境
- 孟庙
- 微山湖
- 水泊梁山
- 太白楼

### AI使用说明（AI Information）

介绍项目中AI技术的应用：

- 内容生成
- 中韩翻译
- 路线规划
- 代码辅助开发
- UI界面设计
